ONE-Data - 介绍

ONE-Data实现了JPA规范，为开发人员提供了功能强大、使用简单的持久化数据管理框架

ONE-Data封装了和关系型数据库交互的操作，让开发人员把注意力集中在编写业务逻辑上

ONE-Data可以作为独立的持久层框架使用，也可以轻松的与其它Java EE应用框架集成

ONE-Data的目标是降低持久层的工作量，让开发人员简单、高效的完成工作

ONE-Data提供了JPA规范和非JPA规范的查询方式，并具备事物处理功能
